

pnpm serve per:
- installare librerie (aws-cdk-lib)
- gestire dipendenze
- lavorare in monorepo
- alternativa a npm

AWS CDK è un tool che ti permette di:
- scrivere codice TypeScript
- creare infrastruttura
- senza usare clic manuali su AWS console web 
CDK crea uno stack chiamato: CDKToolkit composto da:
- bucket S3
- ruoli IAM
- risorse di supporto

CDK deve:
caricare codice Lambda
fare deploy
gestire versioni
